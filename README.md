#markdown 문법의 장점
1. 간결하다
2. 별도의 도구없이 작성가능
3. 다양한 형태로 변환 가능
4. 텍스트로(Text)로 저장되기 때문에 용량이 적어 보관이 용이하다.
5. 지원하는 프로그램과 플랫폼이 다양하다.

#markdown 문법의 단점
1. 표준이 없다.
2. 표준이 없기 때문에 도구에 따라서 변환방식이나 생성물이 다르다.
